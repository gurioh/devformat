Privacy

DevFormat keeps tool input in your browser.

The current tools run client-side. Text you paste into the converters is processed in the browser and is not intentionally transmitted to a DevFormat application server because none exists for these tools today.

What we collect

DevFormat currently uses Google Analytics to understand aggregate page traffic and product usage patterns. This can include page views, device information, and anonymized interaction data handled by Google Analytics.

What the tools process

The tool inputs themselves are processed in client-side JavaScript. Sensitive content should still be handled with care, but the current implementation does not submit text to a dedicated backend for transformation.

Contact and updates

If the product scope changes and server-side processing is introduced later, this page should be updated before those changes are promoted as production behavior.
