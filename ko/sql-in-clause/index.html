FAQ

입력한 텍스트가 서버로 전송되나요?

아니요. 이 도구는 브라우저에서 실행되며 붙여넣은 값은 기기 밖으로 전송되지 않습니다.

빈 줄을 자동으로 제거할 수 있나요?

네. 공백 정리와 빈 줄 제외 옵션을 켜면 불필요한 줄을 제거한 뒤 결과를 만들 수 있습니다.

JavaScript 배열이나 Python 리스트도 만들 수 있나요?

네. 관련 프리셋을 선택하면 구분자, 따옴표, 감싸기 문자가 한 번에 바뀝니다.
