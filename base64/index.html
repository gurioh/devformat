FAQ

Does it support UTF-8 text?

Yes. The page uses TextEncoder and TextDecoder so multi-byte characters survive the encode and decode cycle.

Can I decode a copied token safely?

Yes, as long as the input is valid Base64. Invalid strings are caught and shown as an error instead of crashing the page.

Why use this instead of a CLI?

It is faster for quick inspection when you are already in the browser and only need to convert a small payload or header value.

Useful alongside URL encoding and JSON formatting when debugging payloads end-to-end.
