Share text formatter settings without sharing text

Copy a settings link from DevFormat so you can reopen the same formatter configuration later or send the same output recipe to a teammate without exposing the original input values.

Steps

  1. Open the formatter and set up the output you want.
  2. Use Copy recipe link.
  3. Paste that link into a note, ticket, or message.
  4. Anyone opening it will get the same settings applied, but not the original pasted values.

When this is useful

  • Sharing an agreed formatter recipe with teammates
  • Keeping repeatable setup links for your own future work
  • Documenting exact output rules in runbooks or onboarding docs

Related tools

If you want to save the settings for yourself too, use the recipe actions in formatter. If a simple preset is enough, you can also share settings links from multiline.

FAQ

Does the shared link include my input text?

No. The link contains settings only, such as the preset, delimiter, quote, wrappers, and toggles.

Can I edit the settings after opening a shared link?

Yes. Opening the link applies the shared configuration, and then you can continue adjusting it before copying or saving a recipe.

Can I use the same idea in multiline?

Yes. Multiline and formatter both support settings links, but formatter is better when you want reusable named recipes.
