FAQ

Does this validate JSON as well as format it?

Yes. The validate action parses the input and tells you whether the JSON is valid without changing the structure.

What happens to invalid JSON?

The output panel shows the parsing error message so you can identify where the payload broke.

When should I use minify instead of prettify?

Use minify when you need a compact payload for transport or storage, and prettify when you need to inspect or review the structure.

JSON pages often lead to Base64 or URL inspection next, especially while debugging request and response payloads.
