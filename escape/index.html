FAQ

What does JSON string escaping do?

It converts quotes, backslashes, tabs, carriage returns, and newlines into escape sequences so the value can be embedded safely inside a JSON string.

Can this reverse common escape sequences?

Yes. Unescape mode handles common sequences such as \n, \t, \r, quotes, and backslashes.

When should I use regex-safe mode?

Use it when you need to escape regex metacharacters before matching user-provided text inside a pattern.
