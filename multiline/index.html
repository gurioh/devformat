When to use this page

Use this when you already have raw values on separate lines but need them reshaped to match a code block, query, or copy target. It is also useful when debugging payloads that arrive as one long line and need to be inspected item by item.

FAQ

Does it work both directions?

Yes. Switch the mode to split a single-line string back into separate rows using the configured delimiter, quote character, and wrapper.

Can I use a custom delimiter?

Yes. Choose the custom preset and set the delimiter, quote character, and wrapper fields directly.

Does it handle tabs and newlines?

Yes. You can type \t or \n in the delimiter field and the tool will interpret the escape sequence.
